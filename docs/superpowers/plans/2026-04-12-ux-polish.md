# UX Polish Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Four focused UX improvements: (1) copy PAC URL to clipboard from the TUI dashboard, (2) persist bandwidth history to disk so charts survive restarts, (3) add an `enabled` flag to `PortForward` so forwards can be temporarily disabled without deleting them, (4) desktop notifications when connections drop.

**Architecture:** Clipboard uses Textual's built-in `App.copy_to_clipboard()`. Bandwidth history appends to a JSON file in the workspace on each `_BandwidthSampler` sample and loads it on startup. The `enabled` flag is a new bool field on `PortForward` defaulting `True`; facade's `start()` skips disabled forwards; the TUI connection editor exposes a toggle. Desktop notifications are already wired by the `_notify()` helper added in the core-improvements plan — this plan adds the trigger site in `_ReconnectMonitor`.

**Tech Stack:** Textual `App.copy_to_clipboard()`, Python `json`, Pydantic field addition

- [ ] **Step 9: Add a `toggle_forward_enabled` method to `SusOpsManager`**

```python
    def toggle_forward_enabled(self, conn_tag: str, src_port: int, direction: str) -> bool:
        """Toggle enabled on a forward. Returns the new enabled state."""
        conn = get_connection(self.config, conn_tag)
        if conn is None:
            raise ValueError(f"Connection {conn_tag!r} not found")
        forwards = conn.forwards.local if direction == "local" else conn.forwards.remote
        for fw in forwards:
            if fw.src_port == src_port:
                new_enabled = not fw.enabled
                fw.enabled = new_enabled
                self._save()
                self._emit("forward", {
                    "conn_tag": conn_tag,
                    "src_port": src_port,
                    "direction": direction,
                    "enabled": new_enabled,
                })
                return new_enabled
        raise ValueError(f"Forward {src_port} not found in {conn_tag} {direction}")
```

```python
    def action_toggle_forward(self) -> None:
        """Toggle enabled on the currently selected forward row."""
        focused = self.focused
        if not isinstance(focused, DataTable):
            return
        table_id = focused.id  # "local-table" or "remote-table"
        direction = "local" if table_id == "local-table" else "remote"
        row_key, _ = focused.coordinate_to_cell_key(focused.cursor_coordinate)
        # row_key is the src_port stored as the row key
        try:
            src_port = int(str(row_key.value))
        except (ValueError, AttributeError):
            return
        conn_tag = self._selected_conn_tag()  # existing helper method
        try:
            new_state = self.mgr.toggle_forward_enabled(conn_tag, src_port, direction)
            self.notify(
                f"Forward {src_port} {'enabled' if new_state else 'disabled'}",
                timeout=2,
            )
            self._reload()
        except Exception as exc:
            self.notify(str(exc), severity="error", timeout=3)
```
